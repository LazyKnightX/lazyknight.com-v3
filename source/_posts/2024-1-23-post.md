title: 记录玩家已经阅读的游戏公告、为游戏公告显示红点
date: 2024-1-23 15:12:53 +0800
tags: [游戏UI开发]
---
预设游戏内最多同时展现5个公告，每一个公告的ID不同。
只给每个玩家5个整数储存位。

思路：
1. 获取玩家已经阅读的公告ID
2. 遍历当前游戏内展现的公告ID
3. 将那些没有阅读的公告添加上红点

伪代码：
```
已读 = {}

循环 i = 1, 5:
    id = 读取玩家储存位("已读公告ID", i)
    已读[id] = true

循环 i = 1, 5:
    id = 获取游戏公告ID(i)
    if 已读[id]:
        显示公告红点(id)
```
